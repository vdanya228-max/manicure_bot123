# Telegram-бот для мастера маникюра

Полностью готовый бот на **aiogram 3.x** + **SQLite** + **APScheduler**.

## ✨ Функционал

### Для клиентов:
- Просмотр доступных дат и времени (на 30 дней вперёд)
- Запись на удобное время (только 1 активная запись на пользователя)
- Ввод имени и телефона
- Автоматическое напоминание за 24 часа
- Отмена своей записи
- Обязательная подписка на канал перед записью
- Кнопки **Прайсы** и **Портфолио**

### Для администратора:
- Полная админ-панель (`/admin`)
- Добавление/удаление временных слотов
- Закрытие/открытие дней
- Просмотр расписания на любую дату
- Отмена любой записи клиента
- Статистика

## 🚀 Установка и запуск

### 1. Клонирование / скачивание

Скопируйте все файлы в папку `manicure_bot`.

### 2. Установка зависимостей

```bash
cd manicure_bot
pip install -r requirements.txt
```

### 3. Настройка config.py

Откройте `config.py` и заполните:

```python
BOT_TOKEN = "123456:ABC-DEF..."          # от @BotFather
ADMIN_ID = 123456789                     # ваш Telegram ID (узнать у @userinfobot)
CHANNEL_ID = -1001234567890              # ID канала (с -100)
CHANNEL_LINK = "https://t.me/your_channel"
```

Или создайте файл `.env` в той же папке:

```env
BOT_TOKEN=123456:ABC...
ADMIN_ID=123456789
CHANNEL_ID=-1001234567890
CHANNEL_LINK=https://t.me/your_channel
```

### 4. Запуск бота

```bash
python bot.py
```

Бот автоматически:
- Создаст базу данных `manicure_bot.db`
- Добавит стандартные временные слоты
- Восстановит напоминания при перезапуске

## 📌 Важные моменты

1. **Обязательная подписка** — работает через `getChatMember`.
2. **Напоминания** — планируются только если до записи больше 24 часов. Восстанавливаются при старте бота.
3. **Один пользователь = одна запись** — enforced в БД (`UNIQUE(user_id)`).
4. **При отмене** — слот снова становится доступным + удаляется задача напоминания.
5. **Календарь** — реализован через inline-кнопки с красивыми названиями дат.
6. **HTML-форматирование** — используется везде.

## 🛠️ Полезные команды

- `/start` — главное меню
- `/admin` — админ-панель (только для ADMIN_ID)
